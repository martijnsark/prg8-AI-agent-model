Multi-Tool Email Agent

On this home page, you can see which tools the AI uses and how to connect your own email account to the send_email tool. Follow these steps to set up a Google app password:

  1. Sign in to your Google account and open the security settings.
  2. Turn on 2-Step Verification if it is not already enabled.
  3. Open the app passwords page and create a new app password.
  4. Copy the app password and enter your email address and password on the Send emails page.
  5. Save the settings and test whether the AI can send an email through send_email.

Other tools available to you

  • get_weather: checks current weather for a city.
  • get_current_date: returns today's date in a readable format.
  • roll_dice: rolls a dice with the number of sides you choose.
  • retrieve: finds useful email-writing examples from the document library.
  • send_email: sends an email with your connected account settings.

This connects your email securely to the AI so you can start using the available tools right away.
